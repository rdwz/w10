== Windows 10

.File Explorer
|===

|Hide/restore all windows

|kbd:[Win+E]
|Launch File Explorer

== Chrome

|===
|Shorcut |Purpose

|kbd:[F11]
|Toggle fullscreen

|kbd:[Ctrl+T]
|Open a new tab

|kbd:[Ctrl+Shift+N]
|New incognito window

|kbd:[Ctrl++]
|Zoom in on current page

|kbd:[Ctrl+-]
|Zoom out on current page

|kbd:[Ctrl+0]
|Reset page zoom to default value
|===
